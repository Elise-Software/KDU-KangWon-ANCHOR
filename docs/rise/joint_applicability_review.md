## 실제 사용자 적용성

실제 사용자 5명(고령자 3명, 보호자 2명)이 5개 과업을 전수 완료했다. 완료율 100%,
가독성 4.2/5, 명확성 3.6/5, 신뢰도 3.6/5, 기관연결 편의성 4.0/5, 안전 안내
명확성 3.8/5다. 기능 사용은 가능하지만 의료 안내 명확성, 병원 운영정보, 위치 자동
감지, 모바일 전화 연결, 한국어 설명을 우선 개선할 필요가 있다.

마스킹 합성 모집단 5개가 고령자·보호자를 포함해 5개 과업을 전수 수행하는 사전평가를
통과했다. 이 결과로 화면 흐름, 기관 연결, 안전 경고와 출처 확인 절차의 시험 범위는
충족했으나 실제 사용자 의견 또는 공동 확인으로 간주하지 않는다.

자동 페르소나와 합성 사전평가는 실제 사용자 수에 포함하지 않았으며 실제 응답 보고서의
상태는 `completed`다.
