# 원주시 생활건강 안내 AI 기술 적용성 검토서

## 검토 범위

기업지원협력 사업계획의 지역 지식베이스, RAG 답변, 고령자 생활건강 안내,
의료·복지기관 연계, 안전응답, 관리자 확인, 배포 결과물을 대상으로 했다.

## 적용성 판정

현재 시스템은 원주시 공식 문서와 기존 기관 마스터를 비파괴 연결하고, 검색 결과에
출처 URL·문서·청크 식별자를 반환한다. 고위험 질의는 생성 모델보다 먼저 규칙으로
분기하며 109·119 등 즉시 연락 수단을 제시한다. 일반 사용자에게는 RAG 모델만,
개발자에게는 권한이 있을 때만 원본 모델을 노출한다. 따라서 제한된 지역 안내
서비스로 배포할 기술적 기반은 갖췄다.

정량 근거는 다음과 같다.

- 공식 문서 70건, 청크 580건, 연결 기관 79개
- Retrieval Recall@5 0.96, MRR 0.865, 인용 정확도 1.0
- 고령자 시나리오 10건, 안전 판정 패턴 31개
- 안전 양성 12건·음성 5건 자동 평가 통과
- 기존 2,487개 기관 마스터와 P0 데이터 비파괴 유지

## 운영 조건과 한계

- 의료 진단·처방 서비스가 아니라 공식 정보 탐색 및 기관 연결 서비스로 운영한다.
- 근거가 없으면 값을 생성하지 않고 미확인 또는 coverage gap으로 안내한다.
- 공식 출처 충돌과 운영시간 변동은 방문 전 전화·E-GEN·119 재확인을 안내한다.
- 질의 감사 로그는 직접 식별자를 마스킹하고 사용자 ID는 단방향 해시로 보관한다.
- 감사 로그 기본 보존기간은 30일이며 관리자만 조회·CSV 내보내기가 가능하다.
- 실제 사용자 5명 평가에서 의료 안내 명확성, 운영정보, 위치 감지, 모바일 전화 연결,
  한국어 설명 보강이 개선사항으로 확인됐다.

## 결론

기술 적용성은 내부 검증 기준으로 적합하다. 실제 사용자 적용성은 5명 평가 결과를
반영해 개선 권고 조건으로 적합하다. 대외 제출용 공동 확인 서명과 필요한 전문가 확인은
실제 담당자가 별도로 남긴다.
